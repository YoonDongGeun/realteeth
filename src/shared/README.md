# Shared Layer

프로젝트 전체에서 재사용 가능한 공통 코드를 포함합니다.

## 역할

- 특정 비즈니스 로직에 종속되지 않는 공통 유틸리티
- 재사용 가능한 UI 컴포넌트
- 공통 타입 정의 및 헬퍼 함수

## 하위 디렉토리

### `ui/`

- 재사용 가능한 기본 UI 컴포넌트
- 예: Button, Input, Card, Icon 등

### `lib/`

- 유틸리티 함수
- 헬퍼 함수
- 공통 훅(hooks)

### `api/`

- API 클라이언트 설정
- HTTP 요청 유틸리티
- 인터셉터 및 에러 핸들링

### `types/`

- 공통 타입 정의
- 글로벌 인터페이스

## 의존성 규칙

- ✅ 다른 shared 모듈에만 의존 가능
- ❌ 상위 레이어(entities, features, widgets, app)에 의존 불가
